Responsive Web Design


In this Responsive Web Design Certification, I'm learning the languages that developers use to build webpages: HTML (Hypertext Markup Language) for content, and CSS (Cascading Style Sheets) for design.


First, I created a cat photo app to learn the basics of HTML and CSS. Later, I mastered modern techniques like CSS variables by building a penguin and learned best practices for accessibility by creating a quiz site.


Finally, I acquired the skills to make webpages that respond to different screen sizes by developing a photo gallery with Flexbox and a magazine article layout with CSS Grid.

Developers use HTML and CSS to control the content and styling of a page. And they use JavaScript to make that page interactive.

Front End Development Libraries




Once I were familiar with HTML, CSS, and JavaScript, I'll level up my skills by learning some of the most popular front end libraries in the industry.
